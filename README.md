# Thinker

Plataforma SaaS de IA com assistentes especializados configuráveis via RAG.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **UI:** shadcn/ui
- **Backend:** Supabase (PostgreSQL + Auth + Storage)
- **AI:** OpenRouter + Vercel AI SDK
- **Deployment:** Vercel

## Features

- Chat with specialized AI assistants
- RAG-powered knowledge base
- File upload and analysis (PDF, Excel, Word, CSV, XML, images)
- Audio recording with automatic transcription
- Assistant handoff with context preservation
- Folder-based conversation organization
- Admin dashboard for assistant management

## Deploy on Vercel

The easiest way to deploy is using the [Vercel Platform](https://vercel.com).

Check out the [Next.js deployment documentation](https://nextjs.org/docs/app/building-your-application/deploying) for more details.
